ConfirmKit Demo

Testing the fully Headless, Framework-Agnostic @confirmation-box/angular library components.

1. Basic Confirm (Service)

Simple service call using confirmService.confirm().

2. Directive Usage

Using the [confirmTrigger] structural directive on buttons.

3. Async Action Hook

Dialog with an await handler before triggering logical side effects.

4. Multi-step Flow (Promises)

Demonstrate elegant promise chaining natively without bloated config arrays.

5. Queue Management

Triggers 3 dialogs rapidly to verify Core FIFO queue behavior.

6. Custom UI (Headless Mode)

Trigger a dialog that reads `data.useCustomTheme` to let the ng-template render custom UI.

7. Data Payloads

Pass custom UI config (like images) natively via `data` types.
